# โปรแกรมที่ 1 : EvenOdd.java

## Input
รับตัวเลขจำนวนเต็ม 1 จำนวน

## Process
รับค่าตัวเลขจากผู้ใช้
ตรวจสอบว่าตัวเลขหารด้วย 2 ลงตัวหรือไม่
ถ้าหารลงตัว แสดง "Even number"
ถ้าไม่ลงตัว แสดง "Odd number"

## Output
แสดงผลว่าเป็นเลขคู่ (Even number) หรือเลขคี่ (Odd number)

# วิธีรันโปรแกรม

1. เปิดโปรเจกต์ด้วย Visual Studio Code หรือ Eclipse
2. เปิดไฟล์ Java ที่ต้องการ
3. Compile โปรแกรม

bash
javac EvenOdd.java
javac Score.java
javac ArrayScore.java
javac MinMaxArray.java



4. Run โปรแกรม


bash
java EvenOdd
java Score
java ArrayScore
java MinMaxArray
